# Bank Account Kata
This project is a Spring Boot kata application implementing few features of a Bank Account. It provides RESTfull endpoints to manage deposit, withdrawal and account transactions.

## Prerequisite
- Java 17 
- Maven
- Spring Boot
- Swagger

## Features
- Deposit into account
- Withdraw from account
- Retrieve transaction details
- OpenAPI documentation for easy API testing

## Compile
mvn clean install

## Unit test
mvn test

## Run
mvn spring-boot:run

## Open API (for testing)
http://localhost:8081/swagger-ui.html
